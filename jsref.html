JavaScript Reference:

select(mode)

Select a type of sign

mode-string: one of sign, flag, helmet, football or baseball

setText(top,bottom)

Set the text on the sign

top-string: the first line of text

bottom-string: the second line of text

setOutlineColor(color)

Set the outline color

color-string: hex color or HTML color

setBackgroundColor(color)

Set the background color

color-string: hex color or HTML color

setObjectColor(color)

Set the object color

color-string: hex color or HTML color

setTextColor(color)

Set the text color

color-string: hex color or HTML color

hideObject()

Hide the object
